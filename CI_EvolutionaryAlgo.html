Goals (taken from report):

This project is intertwined between three subgoals. These goals include implementing a convolutional neural network, implementing an evolutionary algorithm for neural architecture search, and evaluating the performance of the algorithm that is created with the aforementioned aspects. This leads to a focus on hyperparameter optimization within the neural architecture search. The aim of the model is to minimize the classification error of images by the neural network. Here, the classification error is measured using the Scikit-Learn Digits dataset.

Experience gained:

Researching about evolutionary algorithms and how to use this for neural architecture search was a valuable experience for me, despite the tight deadline. The deadline was short and it was a challenging task to complete but it was a good opportunity to learn about this topic.

One of the key takeaways from this research was understanding the concept of using evolutionary algorithms to optimize neural network architectures. This approach, known as neural architecture search (NAS), has been shown to be effective in finding high-performing architectures for a given task.

During the research, I learned about different evolutionary algorithms like genetic algorithms and particle swarm optimization, and the ways they can be used for NAS. The research also gave me an insight into the challenges of NAS such as computational complexity and the trade-offs between exploration and exploitation.
