Research Question (taken from report):

How and why do different reward functions influence the performance of an agent's reinforcement learning model during the training phase in the game Snake?

Experience gained:

Creating a reinforcement learning (RL) bot that plays the game Snake using PyTorch was a challenging and informative experience. One of the key takeaways from this project was a deeper understanding of how to use RL algorithms, such as Q-learning, to train an agent to make decisions and improve its performance in a game setting. I also gained experience working with PyTorch, a popular deep learning framework, which allowed me to build and train neural networks.

One of the challenges I faced was determining the appropriate reward function for the agent. It was important to design a function that would encourage the agent to move towards the food while avoiding obstacles, such as the walls and its own tail. Additionally, fine-tuning the hyperparameters, such as the learning rate and discount factor, was crucial for achieving optimal performance.

Overall, this project provided hands-on experience with both game development and RL, and I am excited to apply these skills to future projects.
