Experience gained:

Making a computer vision application in openFrameworks that interacts with the contours of an object was a valuable experience for me. One of the key takeaways from this project was understanding how to use computer vision techniques to track and interact with objects in real-time. Additionally, I gained experience working with openFrameworks, a creative coding framework, which allowed me to create interactive and visually engaging applications.

One of the challenges I faced was learning how to use the openCV library in openFrameworks to detect and track the contours of an object. I had to figure out how to process and analyze the video feed from a camera and extract the contours of an object in real-time.

Another challenge I faced was creating an interactive experience that responds to the contours of the object. I had to think creatively and come up with ways to use the contours information to create an engaging and interactive experience.

Overall, this project was a great opportunity to gain experience in creating interactive computer vision applications and working with openFrameworks. I am excited to apply this knowledge to future projects and continue to explore the possibilities of computer vision and interactive design.
