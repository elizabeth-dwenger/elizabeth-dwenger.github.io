Health app

Early prototype + Solution / 03.02.21

  • Adobe XD health app prototype (Solution)

Scope of work

  • - Designing in Adobe XD and prototyping
  • - Storyboarding, sketching
  • - User testing
  • - Branding, logo design
  • - Teamwork

My main role within this project was to create a visual identity for a health app; as such, I focused on a coherent UI design and brand values.

Solution description:

As a result of the research my team conducted, we concluded that an important aspect of health apps is to make the app easy and convenient so that the user does not have to waste their time. Although contributed less to the ideas of how this should be implemented in terms of actual screens in our app, I did decide on how all of these screens would look.

Experience gained:

I had never used Adobe XD before this project, so I learned a lot about how to make components, storyboarding, scrolling, etc. The team I worked with had very good synergy, which made this project very enjoyable, and we also had different ideas which made this project better than it could have been if I would have completed it on my own.
