Scope of work

  • - Utilizing ML & Monte Carlo Sampling knowledge
  • - LaTeX & writing reports
  • - Teamwork

Goal (modified from our report):

This project aimed to look at the impact of the number of samples, depth, and the number of hidden layers on the success of machine learning bots. The bot whose features are changed is rdeep, a bot that uses Monte Carlo sampling to determine which actions to take. The improved bot is then used to make a data set for machine learning bots to be based on. The success of the bot is measured based on the bot’s performance in Schnapsen, a card game similar to sixty-six.

Experience gained:

The group work was evenly split with this report, and all group members were able to work on each aspect of the project to get an understanding of everything which was done. We each still had specific items which we focused on; for me, this was creating/modifying the bots, and writing the report. I also helped with some of the formattings in LaTeX, which introduced me to how to work with LaTeX.
