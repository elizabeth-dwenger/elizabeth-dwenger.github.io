Animal Shelter Website Redesign

Scope of work

  • - UX & UI design
  • - HTML & CSS
  • - Principals of design
  • - Branding, logo design
  • - Prototyping and testing

For six years, I volunteered at a local animal shelter near my high school, and recently looked at their website. The website is aesthetically unappealing and the design is not user-friendly. As such, I decided one weekend to try to remake the website, and the outcome of this is cataloged here.

Solution description:

Because I only worked on this project only during the weekends for one month (esentially 7 days total), the website only consists of essential web pages. Although this was done due to time constraints, this acted as a challenge in consolidation and also led to a user-friendly website. I also employed minimalistic design choices, which is clearly seen in the color pallet I selected.

Experience gained:

This project taught me about working on web-based projects under strict time constraints and how to create cohesive branding. It also made me more confident with my skills within HTML and CSS.
